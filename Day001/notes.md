# Day 001 Notes

## Key Takeaways
- Python uses dynamic typing - no need to declare variable types
- Use meaningful variable names (snake_case convention)
- type() function helps identify data types

## Common Mistakes to Avoid
- Don't start variable names with numbers
- Avoid using Python keywords as variable names
- Be careful with integer division vs float division

## Practice Ideas
- Create a simple calculator
- Build a unit converter
- Make a personal info display program
