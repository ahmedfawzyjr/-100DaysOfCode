# Day 002 Notes

## Key Takeaways
- // (floor division) returns integer result
- ** is the power operator (not ^)
- Operator precedence: PEMDAS applies
- Logical operators return boolean values

## Operator Precedence (High to Low)
1. **
2. *, /, //, %
3. +, -
4. Comparison operators
5. Logical operators
