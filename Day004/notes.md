## Key Takeaways
- range(n) generates 0 to n-1
- enumerate() provides index and value
- break exits the loop completely
- continue skips to next iteration

## Range Function Variations
```python
range(5)           # 0, 1, 2, 3, 4
range(1, 6)        # 1, 2, 3, 4, 5
range(0, 10, 2)    # 0, 2, 4, 6, 8
range(10, 0, -1)   # 10, 9, 8, ..., 1
```
